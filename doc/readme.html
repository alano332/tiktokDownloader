NVDA TikTok Downloader

A powerful and accessible NVDA addon to download videos from TikTok.

Features

  • Video Download: Download TikTok videos in MP4 format.
  • Watermark-Free Option: Attempt to download videos without the TikTok watermark.
  • Quality Control: Choose from various quality options (Best, 1080p, 720p, 480p, 360p).
  • Smart URL Detection: Automatically grabs TikTok URLs from your browser or clipboard.
  • Background Downloads: Downloads continue in the background.
  • Queue System: Add multiple downloads to the queue.
  • Progress Information: See download speed, ETA, and file size in real-time.
  • Sound Notifications: Audio feedback when downloads complete or fail.
  • Auto-Retry: Configurable automatic retry on failure.
  • Download Statistics: Track your total completed downloads.

Keyboard Shortcuts

  • NVDA+Shift+T: Open TikTok Downloader dialog
  • NVDA+Shift+Control+T: Quick download from clipboard
  • Delete: Remove selected download from list
  • Enter: Open file location for completed download
  • Escape: Close the dialog

Configuration

Settings available in NVDA Menu → Preferences → Settings → TikTok Downloader:

  • Download Folder: Set your default download location.
  • Play Sound Notifications: Enable/disable audio feedback.
  • Auto-Retry Attempts: Number of automatic retry attempts on failure.
  • Try Watermark-Free by Default: Always attempt to download without watermark.
